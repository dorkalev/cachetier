# Cachetier

Multi-tiered cache

## Installation

Add this line to your application's Gemfile:

    gem 'cachetier'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cachetier

## Usage

	require 'cachetier'

	Cachetier
